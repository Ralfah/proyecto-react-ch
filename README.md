# RalfantasticStore

Este fue un proyecto realizado para el curso de React en la plataforma de CoderHouse donde se montó un e-comemrce desde cero.

La idea del contenido de productos sale de lo tradicional y recorre posibles insumos de 'superpoderes' o 'experiencias mágicas'.


---
## Actualización 💪
Se agregó Firebase como Base de Datos para el proyecto.

Se cargaron los productos en dicha plataforma y las órdenes de compra generadas quedarán guardadas aquí mismo.



## Comenzando 🚀

_Para empezar se debe comprender la estructura básica que conlleva la creación de una aplicación en general. En caso que estés interesado en profundizar sobre este proyecto, no dudes en generar una copia en tu Git :D_


## Deployment 📦

_La aplicación corre sobre localhost:3000, súper sencillo y fácil de visualizar. Las posibilidades de seguir creciendo son infinitas_
